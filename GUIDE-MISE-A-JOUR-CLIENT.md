## ⚠️ **IMPORTANT : Sauvegarde Obligatoire**

**AVANT** de faire la mise à jour, vous **DEVEZ** sauvegarder vos données !

### **📁 Méthode 1 : Sauvegarde Automatique (Si disponible)**
1. **Ouvrir** votre GestionPro actuel
2. **Aller** dans le menu "Sauvegarde" (si présent)
3. **Cliquer** "Sauvegarde Complète"
4. **Sauvegarder** le fichier sur le Bureau

### **📁 Méthode 2 : Sauvegarde Manuelle (Recommandée)**
1. **Fermer** GestionPro complètement
2. **Appuyer** sur `Windows + R`
3. **Taper** : `%appdata%\GestionPro`
4. **Appuyer** sur Entrée
5. **Copier** tout le dossier qui s'ouvre
6. **Coller** sur le Bureau
7. **Renommer** : "Sauvegarde_GestionPro_[Date_du_jour]"

## 🚀 **Installation de la Nouvelle Version**

### **Étapes Simples :**

#### **1. 📥 Téléchargement**
- Vous avez reçu le fichier : `GestionPro Setup 2.1.0.exe`
- Taille : environ 300 MB

#### **2. 🔧 Installation**
1. **Double-cliquer** sur `GestionPro Setup 2.1.0.exe`
2. **Cliquer** "Oui" si Windows demande l'autorisation
3. **Suivre** l'assistant d'installation
4. **Garder** le dossier d'installation par défaut
5. **Attendre** la fin de l'installation (2-3 minutes)

#### **3. ✅ Premier Démarrage**
1. **Lancer** GestionPro depuis le Bureau
2. **Se connecter** avec vos identifiants habituels
3. **Vérifier** que toutes vos données sont présentes :
   - Produits ✅
   - Clients ✅
   - Historique des ventes ✅

## 🚨 **En Cas de Problème**

### **Si vos données ne sont pas visibles :**

#### **Solution 1 : Restauration Manuelle**
1. **Fermer** GestionPro
2. **Appuyer** sur `Windows + R`
3. **Taper** : `%appdata%`
4. **Supprimer** le dossier "GestionPro" actuel
5. **Copier** votre dossier de sauvegarde
6. **Renommer** en "GestionPro"
7. **Relancer** l'application

#### **Solution 2 : Import via la Nouvelle Fonction**
1. **Ouvrir** GestionPro 2.1.0
2. **Aller** dans "Sauvegarde"
3. **Glisser** votre fichier de sauvegarde
4. **Choisir** "Remplacer toutes les données"
5. **Cliquer** "Démarrer l'Import"
